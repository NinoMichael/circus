## About

React frontend application for Circus platform.

### Tech Stack

- React 19
- Vite
- TailwindCSS
- Material UI
- Framer Motion
- Chart.js

### Installation

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```

### Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Application pages
├── hooks/          # Custom React hooks
├── services/       # API calls
├── lib/            # Utilities and types
└── layouts/        # Page layouts
```
